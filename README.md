# Women’s World Cup Analytics Dashboard

## Overview

This Power BI dashboard provides comprehensive analysis of Women’s World Cup batting and bowling performance. It delivers interactive insights into player statistics, team performance, scoring patterns, wicket analysis, and match records through dynamic visualizations and KPI tracking.

## Features

* Batting and Bowling Analytics dashboards
* Highest individual score, sixes, fours & hundreds KPIs
* Top run scorers and strike rate analysis
* Country-wise batting performance insights
* Bowling statistics including wickets, economy rate & bowling performance
* Interactive charts, tables, and navigation buttons
* Dynamic filters and user-friendly sports analytics visuals

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling
* Interactive Data Visualization

## Dashboard Insights

* Identifies top-performing batters and bowlers
* Analyzes scoring patterns and wicket trends
* Compares country-wise tournament performance
* Tracks batting efficiency and bowling effectiveness
* Provides detailed cricket tournament analytics and match insights

## Project Purpose

The objective of this dashboard is to simplify cricket performance analysis and provide meaningful tournament insights through interactive and data-driven reporting.
